HTML Table Reference

Table Tags

Tag Name Description
<table> Table Wrapper element for a table.
<thead> Table Head Set of rows that define the table header and/or column headers in a table.
<tbody> Table Body Set of rows that define the core data in a table.
<tr> Table Row Container for rows.
<td> Table Data Container for data cells within rows.
<tfoot> Table Foot Set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Sets how many columns a <td> cell will span; cell width.
rowspan Row Span Sets how many rows a <td> cell will span; cell height.
